🚀 Quick Start

Option 1: PHP Built-in Server (Recommended)

cd rainbow-detergents
php -S localhost:8000

Then open: http://localhost:8000

Option 2: XAMPP/MAMP/WAMP

Place the 'rainbow-detergents' folder in your htdocs directory
